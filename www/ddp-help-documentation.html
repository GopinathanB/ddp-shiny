The source code for this project is available on GitHub.

Usage Notes:
Providing input:

  • The first slider on the left allows you to provide the dose of the drug (in milligrams - mg) provided to the subject. Range = 265 to 325 mg, in steps of 0.1 mg. Default = 315 mg.
  • The second slider on the left allows you to indicate the number of hours that have passed since the dose was given to the subject. Range = 0 to 24 hours, in steps of 1/2 hour. Default = 4 hours
 
Observing output:
  • The first line of output shows the input provided.
  • A plot of the data values are shown as indicated. X-axis is the time since the dose was provided, and Y-axis is the drug concentration for 6 subjects in mg/L.
    The blue line indicates a linear fit, and the red line indicates the hours provided as input.
  • The prediction from the model is also shown (units = mg/L).
